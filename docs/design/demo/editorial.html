Building on the contract

The same prose engine renders sharp editorial, warm jewel-tone, or rounded productivity — set by the loaded adapter.

This paragraph is styled entirely by .fg-prose. Body measure, leading and flow rhythm come from canonical reading tokens, so rendered markdown needs no per-element classes. Links like this one pick up the theme link color.

Reading the token hierarchy #

Four document heading levels carry the outline. Headings switch family and weight per theme, but the rung order never moves:

H1 .fg-article-title → H2 fg-prose h2 → H3 fg-prose h3 → H4 fg-prose h4
